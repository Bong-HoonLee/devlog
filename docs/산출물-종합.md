# 프로젝트 산출물 종합

## 기술 스택

| 영역 | 기술 |
|------|------|
| 프레임워크 | Next.js 15 (App Router) + React 19 |
| 스타일링 | Tailwind CSS v4 + next-themes (다크모드) |
| DB | Supabase PostgreSQL + Prisma ORM |
| 인증 | NextAuth.js v5 (GitHub + Google OAuth) |
| 캐싱/레이트리밋 | Upstash Redis |
| 이메일 | Resend |
| 이미지 저장 | Supabase Storage |
| 에디터 | Tiptap WYSIWYG |
| 검색 | cmdk + PostgreSQL tsvector 전문 검색 |
| 애니메이션 | Framer Motion |
| 모노레포 | Turborepo + pnpm |
| CI/CD | GitHub Actions + Vercel 자동 배포 |

---

## Phase별 산출물

### Phase 1: 기반 구축

- Turborepo + pnpm 모노레포 초기화
- Next.js App Router 프로젝트 생성
- Tailwind v4 + 다크모드 설정
- Supabase + Prisma 연동, DB 스키마 마이그레이션
- 기본 레이아웃 (Header, Footer)

### Phase 2: 어드민 & OAuth

- NextAuth.js v5 GitHub + Google 로그인
- 어드민 레이아웃 + 인증 보호
- 글 CRUD Server Actions
- 헤더 로그인/로그아웃 버튼

### Phase 3: 블로그 UI & 댓글

- 블로그 목록 (PostCard)
- 포스트 상세 (마크다운 렌더링, Shiki 코드 하이라이팅, TOC)
- 태그 시스템 + 태그별 페이지
- 댓글/대댓글 시스템

### Phase 4: SEO & 배포

- Metadata API + 동적 OG 이미지 (`/api/og`)
- sitemap.xml, robots.txt (DB 기반 동적 생성)
- RSS 피드 (`/api/rss`)
- JSON-LD 구조화 데이터
- Vercel 배포

### Phase 5: 에디터 & 콘텐츠

- Tiptap WYSIWYG 에디터
- KaTeX 수식 지원
- Callout 컴포넌트
- 자동 저장 + 예약 발행

### Phase 6: UX 개선

- 시리즈 기능 (CRUD + 네비게이션)
- 페이지네이션
- Cmd+K 검색 (cmdk + tsvector GIN 인덱스)
- Framer Motion 애니메이션 (FadeIn, AnimatedList)

### Phase 7: 소셜 기능

- 좋아요 (useOptimistic)
- 이모지 리액션 (6종)
- 알림 시스템 (NotificationBell)
- 댓글 마크다운 (인라인)
- Resend 이메일 알림 (대댓글)

### Phase 8: 성능 & 인프라

- Upstash Redis 조회수 카운터 (INCR + DB 동기화)
- 레이트 리밋 (sliding window)
- ISR (revalidate = 60)
- GitHub Actions CI 파이프라인

### Phase 9: 확장

- 이메일 구독 (double opt-in)
- 구독 해지 페이지
- About 페이지 리디자인 (포트폴리오)

### Phase 10: 추가 개선

- 코드 블록 복사 버튼
- 관련 글 추천 (태그 기반)
- 공유 버튼 (Twitter + 링크 복사)
- 404 페이지 디자인
- Loading 스켈레톤 (Streaming/Suspense)
- 이미지 드래그 & 드롭 / 붙여넣기 업로드

### Phase 11: 코드 품질 리팩토링

- `lib/config.ts` — 모든 상수 중앙화 (BASE_URL 8곳 → 1곳)
- `lib/auth-utils.ts` — 인증 로직 통합 (5곳 → 1곳)
- `mapPostTags()` — 태그 매핑 통합 (6곳 → 1곳)
- 전체 action/page/component에서 공유 모듈 사용

## DB 모델 (10개)

| 모델 | 설명 |
|------|------|
| `Post` | 블로그 글 (제목, 슬러그, 마크다운, 상태, 조회수) |
| `Tag` | 태그 (이름, 슬러그) |
| `PostTag` | 포스트-태그 다대다 관계 |
| `Series` | 시리즈 (제목, 설명) |
| `User` | 사용자 (OAuth, 역할: user/admin) |
| `Comment` | 댓글 (마크다운, 대댓글 쓰레드) |
| `Reaction` | 이모지 리액션 (댓글별, 유저별 중복 방지) |
| `Notification` | 알림 (대댓글 알림, 읽음 처리) |
| `Like` | 좋아요 (포스트별, 유저별 중복 방지) |
| `Subscriber` | 이메일 구독 (double opt-in, 토큰 인증) |
